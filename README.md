# 📚 Library Management System

A responsive Library Management System built using HTML, CSS, and JavaScript. The application allows users to manage books, search records, update book status, and maintain a clean digital inventory through an intuitive interface.

## 🚀 Features

- Add new books
- Search books by title, author, or ID
- Edit existing books
- Delete books
- Track Available and Borrowed books
- Dashboard statistics
- Responsive layout

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
